Source code download

Beta version

The current Beta release is 2.5 and avaliable for download here or from the google downloads section.

Stable version

At present there are no "stable" releases made since the Quadrics 2.2 version, a release is scheduled for early June 2009.

Quadrics release

The latest stable release from Quadrics is 2.2 was made on the 16th March 2009 and at the time of writing is downloadable from their website in the QSNET2LIBS package. The padb avaliable on this site is a fork of that release.

Source code access

Subversion access to padb is hosted by google where you can browse the source online.

Alternatively to checkout a read-only copy of the source run the following command from your desktop. 

svn checkout http://padb.googlecode.com/svn/trunk/ padb-read-only
